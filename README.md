# Lightning Pay - Generador de Enlaces Bitcoin ⚡

## Descripción
Página web simple y moderna para crear enlaces de pago Lightning Network. Convierte euros a satoshis y genera QR codes para pagos fáciles.

## Características
- Conversión EUR a SATS en tiempo real (aprox)
- Generación de enlaces `lightning:`
- Código QR escaneable
- Diseño responsive y dark theme

## Cómo desplegar
1. Habilita GitHub Pages en Settings > Pages.
2. Accede en `https://cristiancgrd13-cpu.github.io/bitcoin-invoice-generator/`

Personaliza agregando tu logo o integrando backend real para invoices válidos.
